# proTankr

**proTankr** is a web-based tool for bulk liquid drivers and operators to plan loads, manage terminal access, and track time-based credentials â€” all in one place.

The app focuses on **pre-scale load planning**, **terminal access tracking**, and **expiration-aware workflows** to reduce friction at the rack and improve day-to-day efficiency.

---

## What proTankr Does (Current)

- Select terminals by state and city
- Track terminal access with activation and expiration dates
- Automatically calculate expiration based on terminal renewal rules
- Highlight expired or expiring access with countdowns
- Maintain a personal **My Terminals** list per driver
- Manage terminal access via an explicit **Get Carded** workflow
- Plan loads using a calculator designed for bulk liquids
